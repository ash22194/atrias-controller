Initial ATRIAS setup

Date: 4/17/2013

INSRUCTIONS
1. Install MATLAB with xPC Target on host Windows machine (64 or 32 bit)
2. Install EtherCAT Configurator ET9000 on target Windows machine (32 bit only)
3. Verify that network cards on both machines are compatible with xPC target AND EtherCAT
4. Connect machines with cross-over cable
5. Connect EtherCAT network and generate ENI (.xml) file using ET9000 software.
6. Follow xPC target setup tutorial (in current folder)
7. Verify calibration parameters in DAQparams.m
